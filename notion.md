## 🛠 프로젝트 핵심 내용 (Key Highlights)

### 1. 지능형 페이지 감지 및 정밀 분할
- **가로/세로 비율 자동 감지**: 모든 페이지를 일률적으로 자르는 대신, 페이지의 `MediaBox` 좌표를 분석하여 가로가 긴 페이지(Landscape)만 선택적으로 분할합니다. 세로형 페이지(Portrait)는 원본을 유지하여 데이터 손상을 방지합니다.
- **여백 문제 해결**: `MediaBox`와 `CropBox`를 동기화하여 분할 후 발생하던 불필요한 흰색 여백 문제를 완벽히 해결했습니다.

### 2. 자동화 및 중복 처리 로직
- **일괄 처리(Batch Processing)**: `PDF/` 디렉토리 내의 모든 파일을 자동으로 읽어 개별 폴더별로 결과물을 생성합니다.
- **중복 작업 방지**: `Splitter/` 디렉토리 내에 동일한 결과 폴더가 이미 존재할 경우 작업을 건너뛰어 시스템 자원을 효율적으로 관리합니다.

---

## 🚀 재현 방법 (Steps to Reproduce)
1. **환경 구축**: `python3 -m venv venv` 및 `source venv/bin/activate`
2. **패키지 설치**: `pip install PyPDF2`
3. **실행**: `PDF/` 폴더에 원본 파일을 넣고 `python split_pdf.py` 실행
4. **결과 확인**: `Splitter/` 폴더 하위에 파일명과 동일한 이름의 폴더 내에 분할된 PDF 생성
